# Blackjack 🃏

## Demo

Open index.html in your web browser to play.

## Features

- Play Blackjack directly in the browser
- Start a new game and draw new cards
- Player has name and chips
- Dynamic messages based on game status
- Simple and responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repo:
   `git clone <your-repo-url>`
2. Open `index.html` in your web browser.

## Usage

- Click "START GAME" to begin.
- Click "NEW CARD" to draw a new card.
- Follow the messages to see if you get Blackjack or bust.
